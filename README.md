# C语言课程大作业 —— 指令解释器

## 一、项目简介
本项目为 C 语言课程大作业，实现了一个简易的指令解释器，
支持从 **文本文件** 和 **二进制文件** 中读取程序并执行。
程序能够完成基本指令解析、执行控制，并对输入文件的合法性进行检查。

## 二、功能说明
- 支持文本格式程序读取与执行
- 支持二进制格式程序读取与执行
- 指令合法性检查与错误处理
- 支持数据区与代码区分离
- 程序运行过程可调试、可验证

## 三、拓展功能实现情况
- **拓展二**：  
  对二进制文件格式进行完整合法性检查，能够识别文件截断、非法长度等错误情况
- **拓展三（方案 A）**：  
  使用动态数组存储指令，支持超过 255 条指令的程序执行，
  通过在读取阶段动态扩容指令数组，运行阶段不再受固定大小限制。
